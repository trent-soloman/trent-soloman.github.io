Explain control flow and loops using an example process from everyday life, for example 'waking up' or 'brushing your teeth'.

A while loop is like you keep biting and chewing your toast until it is all gone. A For loop is like where you keeping brushing your teeth until a certain amount of minutes has elapsed.

Explain the difference between accessing data from arrays and object literals.

For an Array to access data we use [ ] eg pets[0] . To access data from object literals we can use literal notation eg birthdays.james

Explain what functions are and why they are useful.

Functions are lines of code that are executed when called upon and save us from typing the same section of code over and over. They may take inputs and return outputs.
