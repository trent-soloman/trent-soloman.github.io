Can you think of an analogy to describe HTML, CSS and the DOM?

The html is like the building and the css is like the interior decoration. The Dom is like a tree where you can pluck the leaves off and rearrange them in a different pattern.

What is meant by boxifying design?

Everything on the page can be broken up into different boxes. Sometimes there are boxes within boxes.

What is the box model?

The CSS box model is a box that wraps around each HTML element. It consists of margins, borders, padding, and the actual content.
