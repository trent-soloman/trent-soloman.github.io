What is grid based design?

An invisible grid can be used to help place elements on a page to help direct our eyes to content, prioritise content and maximise visual appeal

Whats all the hype about responsive webpages?

It provides optimal viewing across a range of devices.

What is semantic structure?

It is the use of HTML markup to reinforce the meaning of information on webpages rather than defining its presentation.
