Using the JS Array docs as a starting point, describe what these functions do in your own words:

The map() method creates a new array with the results of calling a provided function on every element in this array. The filter() method creates a new array with all elements that pass the test implemented by the provided function. The reduce() method applies a function against an accumulator and each value of the array (from left-to-right) to reduce it to a single value.
